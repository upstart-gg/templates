# Minimal template

## Purpose
This template is meant to be a clean slate for building your app without any unnecessary code or features.
It includes only the essential files and configurations needed to get started.

## Contents

### Routes
- A layout (`app/routes/_layout.tsx`) ofr the homepage and future routes
- A homepage (`app/routes/_default._index.tsx`) displaying a skeleton page (should be removed or edited to implement the actual homepage)
- A 404 page (`app/routes/$.tsx`) - Don't edit or remove this file

### Components
- A `SkeletonTemplate` component that is used to display a temporary skeleton page while using the app for the first time (should be removed once the homepage is implemented)

### Themes
- A light and a dark theme (`app/config/themes/default-light.json` and `app/config/themes/default-dark.json`)

---

**IMPORTANT**: This template is meant to be a starting point. You should update or remove any code that is not needed and implement the actual features of your app.
